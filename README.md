# VRDebugger

This package is intended as a helpful window for use within scenes in VR projects. You can purchase and use the package, but please be aware that it is still under development.

Note: This package was developed in the Meta XR environment. It will not work with Open XR projects.

# Current Features

- Console
- Stats(Only FPS Show)
